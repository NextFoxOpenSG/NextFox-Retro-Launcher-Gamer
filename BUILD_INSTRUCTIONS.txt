=== BUILD INSTRUCTIONS FOR NEXTFOX RLG ===

1. BUILD ON WINDOWS (.exe portable)
----------------------------------------
Open a terminal/CMD in the folder where main.py is and run:

   pip install PySide6 pyinstaller
   pyinstaller --noconfirm --onefile --windowed --name "NextFoxRLG" main.py

Result: A "dist/" folder will be created with the file NextFoxRLG.exe.
That executable is 100% portable and works on any Windows PC without installing Python.


2. BUILD ON LINUX (Portable Binary)
-----------------------------------------
In the Linux terminal (e.g. Ubuntu, Zorin OS, Fedora, Bazzite):

   pip install PySide6 pyinstaller
   pyinstaller --noconfirm --onefile --windowed --name "NextFoxRLG" main.py

Result: In "dist/" you will get the portable binary 'NextFoxRLG'.
To run it on any Linux system:
   chmod +x dist/NextFoxRLG
   ./dist/NextFoxRLG
