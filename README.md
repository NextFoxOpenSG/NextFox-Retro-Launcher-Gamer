**NextFox RLG** It is a launcher, installer and executor of video game console emulators for Windows and Linux PCs. The project will seek to integrate the most stable and modern emulators to facilitate search, installation and gameplay. It is recommended to use legally purchased and obtained ROMs to avoid legal and malware-related issues

---

## ✨ Key Features

* **🚀 100% Portable:** Requires no prior installation of Python or system dependencies on the end-user's machine.
* **🕹️ Automatic Emulator Management:** Upon pressing **"STARS"**, the application autonomously downloads and configures the required portable emulators.
* **🐧 Desktop Integration for Linux:** Includes an auto-installation script that integrates the app directly into the system menu with its corresponding `.desktop` shortcut and icon.
* **🎨 Intuitive UI:** Built with **PySide6 (Qt)**, delivering a smooth experience seamlessly integrated with dark themes.
